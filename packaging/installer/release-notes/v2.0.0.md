# Personal Knowledge Base v2.0.0

## Version position

`v2.0.0` is the first Windows installer baseline for Personal Knowledge Base. This release is a local-first personal knowledge base foundation focused on safe install, first-run workspace selection, read-only browsing/search, and user data preservation.

## Included capabilities

- Windows installer based on Inno Setup 6 and the PyInstaller one-folder output.
- PySide6 read-only GUI.
- Select an existing knowledge-base workspace.
- Create a minimal knowledge-base workspace after plan preview and explicit confirmation.
- Search formal knowledge.
- Browse the knowledge library.
- Preview and read a single document.
- View task center summary.
- View read-only settings and local paths.
- Remember window size and position.
- Application icon and basic branding.
- Preserve user data across uninstall and reinstall.

## Data safety

- User workspaces are not stored in the install directory.
- GUI settings and logs are stored under `%LOCALAPPDATA%\PersonalKnowledgeBase\`.
- Uninstall does not delete user workspaces.
- Uninstall does not delete LocalAppData settings or logs.
- Reinstall does not delete user workspace data.
- Reinstall does not delete LocalAppData settings or logs.
- Empty workspace startup does not create `.kb`.
- The app does not automatically index.
- The app does not automatically import files.
- Git is optional and not required for ordinary installed-app use.
- Python is not required for ordinary installed-app use.

## Not included

- AI assistant.
- RSS.
- Vector search.
- Mutation UI.
- Archive/delete/merge/template apply/restore/promote execute.
- Auto update.
- Code signing.
- One-file packaging.

## Known limitations

- The installer is unsigned, so Windows may show a security warning.
- The app is still packaged as one-folder content installed by Inno Setup, not as a one-file executable.
- Auto update is not included.
- AI is not included.
- The user still needs to select or create a workspace.
- Screenshot-level visual verification of the double-click installer wizard and unsigned warning remains a P3 post-release polish item; automated install, launch, uninstall, reinstall, shortcut, and data-preservation checks passed.

## Upgrade and reinstall

- Reinstalling does not delete user workspaces.
- Reinstalling does not delete LocalAppData settings or logs.
- To fully remove local user data, the user must manually delete the selected workspace and `%LOCALAPPDATA%\PersonalKnowledgeBase\`.
