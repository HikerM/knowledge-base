# Personal Knowledge Base v2.0.0-rc.2

## Version position

`v2.0.0-rc.2` is a common-user acceptance polish candidate for the Windows installer. It builds on the `v2.0.0-rc.1` installer spike and is still not the final public release.

## What changed since rc.1

- Added a common-user acceptance report for install, launch, workspace selection, workspace creation, uninstall, reinstall, and user data preservation.
- Revalidated Start Menu shortcut creation.
- Revalidated optional desktop shortcut creation.
- Aligned user-visible version text and packaging metadata to `v2.0.0-rc.2`.
- Updated installer and packaging documentation to describe the rc.2 acceptance baseline.

## Included capabilities

- PySide6 read-only GUI.
- First-run Workspace Gate.
- Select an existing workspace.
- Create a minimal workspace after plan preview and explicit confirmation.
- Search formal knowledge.
- Browse formal knowledge.
- Preview and read a single document.
- View task center summary.
- View read-only settings and local paths.
- Remember window size, position, and last workspace.
- Application icon and basic branding.
- Windows installer based on the PyInstaller one-folder output.

## Data safety boundary

- User workspaces are not stored in the install directory.
- GUI settings and logs are stored under `%LOCALAPPDATA%\PersonalKnowledgeBase\`.
- Uninstall does not delete user workspaces.
- Uninstall does not delete LocalAppData settings or logs.
- Reinstall preserves workspace data and LocalAppData settings/logs.
- Empty workspace startup does not create `.kb`.
- The app does not automatically index.
- The app does not automatically import files.
- Git is optional and not required.
- Python is not required for the installed app.

## Not included

- AI assistant.
- RSS.
- Vector search.
- Mutation UI.
- Archive/delete/merge/template apply/restore/promote execute.
- Code signing.
- Auto update.
- One-file packaging.

## Known limitations

- The installer is still unsigned, so Windows may show a security warning.
- The app is still packaged as one-folder content installed by Inno Setup.
- Auto update is not included.
- AI, RSS, vector search, and mutation UI remain out of scope.
- Workspace import wizard is not included.
- A final human visual pass is still recommended for the double-click wizard and unsigned-installer warning screenshots before `v2.0.0 final`.

## Upgrade and reinstall

- Reinstalling does not delete user workspaces.
- Reinstalling does not delete LocalAppData settings or logs.
- To fully remove local user data, the user must manually delete the selected workspace and `%LOCALAPPDATA%\PersonalKnowledgeBase\`.
