# Personal Knowledge Base v2.0.0-rc.1

## Version position

`v2.0.0-rc.1` is the first Windows installer release candidate for Personal Knowledge Base. It is still a candidate build, not the final public release.

## New capabilities

- PySide6 read-only GUI.
- Select an existing workspace.
- Create a minimal workspace after plan preview and explicit confirmation.
- Search formal knowledge.
- Browse the knowledge library.
- Preview and read a single document.
- View task center summary.
- View read-only settings information.
- Remember window size and position.
- Application icon and basic branding.
- Windows installer based on the PyInstaller one-folder output.

## Data safety

- User workspaces are not stored in the install directory.
- GUI settings and logs are stored in LocalAppData.
- Uninstall does not delete user workspaces.
- Uninstall does not delete LocalAppData settings or logs.
- The app does not automatically index.
- The app does not automatically import files.
- Git is optional and not required.

## Not included

- AI assistant.
- RSS.
- Vector search.
- Mutation UI.
- Archive/delete/merge/template apply/restore/promote execute.
- Code signing.
- Auto update.

## Known limitations

- The installer is currently unsigned, so Windows may show a security warning.
- The app is still packaged as one-folder, not one-file.
- Auto update is not included.
- AI is not included.
- Workspace import wizard is not included.
- The user may need to manually select a workspace.

## Upgrade and reinstall

- Reinstalling does not delete user workspaces.
- Reinstalling does not delete LocalAppData settings or logs.
- To fully remove local user data, the user must manually delete the selected workspace and `%LOCALAPPDATA%\PersonalKnowledgeBase\`.

## Acceptance commands

```powershell
powershell -ExecutionPolicy Bypass -File packaging\installer\inno\installer_smoke.ps1
python scripts\kb.py audit
python scripts\kb.py secret-scan
python tests\gui_smoke_test.py
```

Full local release-candidate validation should also run the baseline service, GUI, packaging, and governance tests listed in the repository README.
