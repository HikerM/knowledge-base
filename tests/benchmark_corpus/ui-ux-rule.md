# Adaptive Dashboard Density Rule

Operational dashboards should use restrained density: compact controls, clear grouping, and stable table dimensions.

## 可执行规则

- Prefer dense but readable rows over decorative card-heavy layouts.
- Keep dashboard filters close to the data table they affect.
- Avoid oversized hero typography inside repeated operational panels.

## 验证方式

Review desktop and tablet screenshots for scan speed, row alignment, and text overflow.
